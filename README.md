# WWE Universe - Hall of Fame & Current Superstars

A tribute website celebrating WWE Hall of Famers and current superstars with modern animations and responsive design.

## Live Demo

Visit: `https://sandeepbikram.github.io/MyRepo/`

## Features

- Preloader animation with spinning rings
- Floating particle background
- Scroll-triggered animations (Intersection Observer API)
- 3D card tilt effect on hover
- Counter animation for stats
- Mobile responsive with hamburger menu
- Smooth scroll navigation
- Active nav link highlighting
- Back-to-top button
- Image fallback placeholders

## Setup

1. Clone the repository
2. Add wrestler images to the `images/` folder (or use placeholder fallbacks)
3. Push to GitHub
4. Enable GitHub Pages in Settings > Pages > Source: main branch

## Image Guidelines

- Recommended size: 400x450px or similar portrait ratio
- Format: JPG or PNG
- The site includes fallback placeholders if images are missing

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (No frameworks)
